# 👨🏻‍🎓 Student API

> A mock API using Hono, Bun, Neon

## First time setup

### Set up Neon connection

- Create a `.env.local` file based on `.env.example`

### Create table in Neon DB

```
bun run generate
bun run migration
```

### Running the app

```
bun i
bun run start
```
